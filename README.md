# 欣聯大心社區委員輪值抽籤系統

社區管理委員會年度輪值抽籤網頁。

## 🔗 線上網址

👉 [https://wenbatman33.github.io/committee_duty_roster/](https://wenbatman33.github.io/committee_duty_roster/)

## 📋 功能說明

- **三棟大樓住戶資料**
  - A棟：17戶（A1-1F，以及 A1、A2 各 2F~9F）
  - B棟：52戶（B1~B4 各 2F~14F）
  - C棟：38戶（C2、C3 在 2F；C1~C3 各 3F~14F）

- **抽籤功能**
  - 點擊「開始抽籤」進行隨機排序
  - 動畫效果顯示抽籤過程
  - 大樓圖示上顯示每戶的輪值號碼

- **結果儲存**
  - 使用 LocalStorage 儲存抽籤結果
  - 重新整理頁面後自動載入上次結果

- **輪值表格**
  - 顯示完整輪值順序
  - 可列印輸出

## 🏗️ 技術

純靜態網頁，使用 HTML / CSS / JavaScript 開發，部署於 GitHub Pages。
